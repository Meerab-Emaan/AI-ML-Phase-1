# AI-ML-Phase-1
Machine Learning Lab Tasks using Python, Pandas, Scikit-Learn, Matplotlib, and Seaborn. Includes Iris Dataset Exploration, Stock Price Prediction using Yahoo Finance Data, and Heart Disease Prediction using Classification Models.

**Machine Learning Lab Tasks**

This repository contains the machine learning tasks completed during my internship. The projects focus on data analysis, visualization, and predictive modeling using real-world datasets. The work includes exploring and visualizing the Iris dataset, predicting stock prices using historical market data, and developing a heart disease prediction model. Through these tasks, I gained hands-on experience with data preprocessing, exploratory data analysis (EDA), machine learning algorithms, model evaluation, and data visualization using Python and its data science libraries.
